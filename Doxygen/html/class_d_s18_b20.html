Detailed Description

DS18B20.

This class includes functions for reading the 64-BIT lasered ROM code of the DS18B20 Temperature sensor. And uses this 64-BIT Lasered ROM code for applications. For example: a Key. The DS18B20 uses the one wire protocol so it only uses one data wire The one wire pin is defined as "datapin". DS18B20 Datasheet: https://datasheets.maximintegrated.com/en/ds/DS18B20.pdf.
